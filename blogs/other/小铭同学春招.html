    小铭同学春招日志

    Xiao_Ming 5/17/2021 面经
    笔者是双非院校大三的学生，总共得到了阿里、百度、滴滴、网易、京东、小米(非时间顺序)前端实习的认可。
现，在阿里实习。

    ​ 笔者与大二下开始自学的前端，大二下之前一直在学习后端相关的知识。最后因为一个小小的机遇和朋友鼓励，并且因为自身的兴趣。开始学了前端。

    ​ 我自己春招准备的并不多，因为之前寒假的时候就有在实习，正好春招前那段日子很忙，于是就没能好好准备，很庆幸自己春招还能有所得。于是准备把这一段日子以来的经历总结一下，给这段回忆希望能留下些什么。

    ​ 感谢有全世界最好的导师的指导，感谢无论多么黑暗的日子小王老师都不离不弃的相伴，感谢父母始终能毫无保留的支持，感谢朋友在这条志同道合的路上守望相助，感谢一路上身边陪伴与支持的人和那些虽素未谋面却一直真心相助的人，谢谢大家❤️

    # 面经

    其实之前没有一一总结面经，而且很多公司的面试各种穿插着，所以笔者以公司为节点来记录，公司内以部门为节点来记录。

    # 阿里达摩院小蜜 3 轮--直通终面

    # 一面(电话 1h20min)

    面试约的是晚上 7:30,感觉小蜜是不是很忙 ​ 啊 ​，面试官迟到了足足 40min，迟到 10min 左右的时候找了内推人，内推人说面试官有十分要紧的事情，但今天晚上一定会面的让我耐心等待。

    虽然迟到了面试官人很好，对我一个学生解释了很多很多，自己因为什么事情推迟了面试而且再三表达了歉意。我也完全能理解哈哈，因为出 bug 紧急修复去了

    • 自我介绍
    • 笔试 30min 平台使用的是阿里的伯乐平台，面试官全程看着我写，感觉的到他工作了一天很累...说话什么的很乏，他说他闭麦了休息一下，会看着我，让我好好写。共 4 题，全部都是场景题难以描述就说知识点了。封装 ajax、科学计数法点“，”、深拷贝、扁平化。
    • 三列布局
    • 水平垂直居中
    • BFC
    • float 和 flex 还有 grid 一些对比和思考，以及场景
    • 闭包
    • 了解异步吗？异步的发展过程，现代的 js 异步方案，为什么要这样？
    • Promise，async/await。优缺点？async/await 和 promise 的缺点是什么?
    • Symbol 和 Bigint
    • 原型链了解吗？

    个人感觉小蜜的一面还是比较基础的，反问环节中问了很多小蜜的业务，因为确实不太了解这个部门。

    # 二面(电话面 1h)

    这次面试面到最后才发现面试官我是认识的(是个早 talk 大佬)我有看过她的很多技术分享。可惜大佬不晓得我是她的小粉丝哈哈哈，不然不知道能不能放个水。二面应该是压力面了，问的十分的深和难

    • 说说你的实习经历
    • 你觉得实习成长了什么？
    • 为什么选择前端？
    • 看你还参加了阿里的前端练习生计划，对前端智能化了解吗?
    • 说说你的项目
    • 路由实现方式
    • Hybrid 架构中的通信机制
    • JSBrige 的实现方案
    • 谈谈你对全链路打点中全链路的理解
    • 打点上报的实现？优化？客户端打点和前端打点的区别？
